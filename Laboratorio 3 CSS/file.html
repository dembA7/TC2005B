1. Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS? Creo que "!important" es una herramienta que puede tener usos muy importantes al momento de darle diseño a una página web usando CSS.

2. Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado? Porque el color del fondo puede afectar la legibilidad del contenido.

3. Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt? Los porcentajes pueden usarse cuando se quiere que un elemento se ajuste automáticamente al tamaño del contenedor. El uso de píxeles es adecuado cuando se quiere un control preciso del tamaño. Los puntos son una buena opción para establecer el tamaño de fuente de los elementos.

4. ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio? Porque al eliminar los espacios, el tráfico que llega a la red es menor, dismuyendo tiempos de carga.
